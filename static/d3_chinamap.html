GeoJSON介绍

GeoJSON是一种对各种地理数据结构进行编码的格式。

GeoJSON对象可以表示几何、特征或者特征集合。

GeoJSON支持下面几何类型：点、线、面、多点、多线、多面和几何集合。

GeoJSON里的特征包含一个几何对象和其他属性，特征集合表示一系列特征。

参考：地理空间数据格式——GeoJSON
参考：Geojson笔记一：深度入门
GeoJSON格式规范说明

点的数据格式

{
    "type": "Feature",
    "geometry": {
        "type": "Point",
        "coordinates": [125.6, 10.1]
    },
    "properties": {
        "name": "Dinagat Islands"
    }
}
type指的是数据类型，它是一个feature。它包含几何要素geometry和属性properties。几何要素包含要素的类型和坐标。属性可以包含各种数据类型。

线的数据格式

{
    "type": "LineString",
    "coordinates": [
        [100.0, 0.0], [101.0, 1.0]
    ]
}
线状数据由两个以上的点组合而成，在坐标的外面多包上一层中括号，表示json中的数组。

面的数据格式

{
    "type": "Polygon",
        "coordinates": [
            [ [100.0, 0.0], [101.0, 0.0], [101.0, 1.0], [100.0, 1.0], [100.0, 0.0] ],
            [ [100.2, 0.2], [100.8, 0.2], [100.8, 0.8], [100.2, 0.8], [100.2, 0.2] ]
        ]
}
由于面状数据可以为镂空多边形，可以支持一个以上的多边形，所以在线状数据的基础上多加一层中括号。

特征集合FeatureCollection

{ 
    "type": "FeatureCollection",
    "features": [
        {
            "type": "Feature",
            "geometry": {
                "type": "Point", 
                "coordinates": [102.0, 0.5]
            },
            "properties": {
                "prop0": "value0"
            }
        },
        {
            "type": "Feature",
            "geometry": {
                "type": "LineString",
                "coordinates": [
                    [102.0, 0.0], [103.0, 1.0], [104.0, 0.0], [105.0, 1.0]
                ]
            },
            "properties": {
                "prop0": "value0",
                "prop1": 0.0
            }
        },
        {
            "type": "Feature",
            "geometry": {
                "type": "Polygon",
                "coordinates": [
                    [ 
                        [100.0, 0.0], [101.0, 0.0], [101.0, 1.0], [100.0, 1.0], [100.0, 0.0] 
                    ]
                ]
            },
            "properties": {
                "prop0": "value0",
                "prop1": {
                    "this": "that"
                }
            }
        }
    ]
}

用D3JS绘制中国地图

参考：d3js官方文档

定义投影函数

var projection = d3.geoMercator()
                    .center([110, 40])
                    .scale(700);

.mercatory()指定投影方式，d3.geo提供多种投影方式。

  • d3.geo.albersUsa()阿伯斯投影：用于美国地图，可以把阿拉斯加和夏威夷显示到美国地图旁边。
  • d3.geo.azimuthalEqualArea()等面积方位投影：适合等值线图，这个投影的极坐标方向被用来作为联合国图标。
  • d3.geo.conicConformal()圆锥共形投影：兰伯特的圆锥共形投影将地球投影在一个圆锥上。
  • d3.geo.conicEqualArea()圆锥等面积投影：阿伯斯投影，作为一种等面积投影，被推荐作为等值线图，因为它保留了相对区域的地理特征。
  • d3.geo.conicEquidistant()圆锥等距投影
  • d3.geo.equirectangular()相等矩形投影，或者叫普拉特方形投影（貌似这是法国的叫法），是最简单可能的地理投影：标记功能。
  • d3.geo.gnomonic()球心投影：是一种方位投影，它连续不断地投影一个巨大的包围圈。这个网站就是一个例子（你可以想象一下从地球仪中心看地球地图的感觉）。
  • d3.geo.mercator()墨卡托投影：映射平铺的数据时是最常用的，例如用墨卡托投影显示栅格，例如http://bl.ocks.org/mbostock/4150951这个页面。
  • d3.geo.orthographic()正射投影:也是一种方位投影，它适合显示一个半球：角度无穷大。
  • d3.geo.stereographic()极射赤平投影是另一个角度的方位投影。它的视角相当于站在地球的表面向里面看（跟球心投影相反），它因此被经常用于天体图。
  • d3.geo.transverseMercator()横向墨卡托投影

.center([107, 31])用于指定地图中心，[107, 31]是中心的经纬度。

.scale(850)设定放大的比例。

.translate()指定平移。

创建path函数

var path = d3.geoPath()
            .projection(projection);

用于接收geoJSON中的经纬度数据，并按照projection中指定的方法转换为.path()绘制path方法所能接受的路径点

读取geoJSON中的数据

d3.json("china.json", function(error, root){}

d3.json(url,function)是用于读取json数据的函数。注意，它只能通过http协议获得数据，所以数据文件必须保存在服务器上。比如static静态文件夹中。

在d3.json()中指定绘图方法

d3.json("china.json", function(error, root){
        if (error)
            return console.error(error);
        console.log(root.features);
        // 获得geoJSON中的features字的数据，使用.path()绘制各省地图
        svg.selectAll("path")
            .data(root.features)
            .enter()
            .append("path")
            // 指定线条颜色
            .attr("stroke", "#000")
            // 指定线条宽度
            .attr("stroke-width", 1)
            // 指定填充色
            .attr("fill", "#FF8400")
            // 指定path的路径点
            .attr("d", path)
            // 绑定鼠标移入时的事件
            .on("mouseover", 
                function(d, i){
                    d3.select(this)
                        // 绑定缓动动画效果，时间为0.5秒
                        .transition()
                        .duration(500)
                        .attr("fill", "red")
                        .attr("stroke-width",2);})
            // 绑定鼠标移出时的事件
            .on("mouseout", function(d, i){d3.select(this).transition().duration(500).attr("fill", "#FF8400").attr("stroke-width", 1)});
    });

svg图形自适应窗口大小

<div class="container" style="width:80%">
    ...
</div>

使用div.container包装svg，以指定svg的大小

var svg = d3.select(".container")
            .append("svg")
            .attr("preserveAspectRatio", "xMidYMid meet")
            .attr("viewBox", "0 0 400 400")

使用preserveAspectRatio和viewBox指定svg的属性，代替通常的width和height
